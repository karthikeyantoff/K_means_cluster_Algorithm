ğŸ“Œ About the Project

This project uses K-Means Clustering, an unsupervised machine learning algorithm, to analyze and group countries based on global development indicators.

Unlike supervised learning models, K-Means does not require labeled data. Instead, it identifies hidden patterns and similarities within the dataset by grouping countries with similar characteristics into clusters.

The Streamlit application provides a simple and interactive interface to explore these clusters visually and gain insights into global development trends.

The project is deployed using Streamlit Cloud.
